# Json 类

一种数据表示结构。

命名空间：[Neo.SmartContract.Framework.Services.Neo](../neo.md)

程序集：Neo.SmartContract.Framework

## 语法

```c#
public static class Json
```

## 属性

| 名称                                       | 说明                         |
| ---------------------------------------- | -------------------------- |
| Serialize(object obj) | 将对象元素序列化为Json串  |
| Deserialize(string json)        | 将Json串反序列化为对象元素                  |

## 构造方法

Json 类是静态类，无需构造方法。
